# Leetcode 151 反转字符串中的单词
https://leetcode.cn/problems/reverse-words-in-a-string

## 方法一 从后往前读取单词
从后往前遍历字符串，把单词从后往前放入新字符串中。
### 易错点
1.新字符串需要动态分配。  
2.空字符串单独判断。  
### 代码
```c
#include <string.h>
#include <ctype.h>
#include <stdlib.h>

char* reverseWords(char* s) {
    int len = strlen(s);
    // 动态堆内存，不会随函数销毁
    char* t = (char*)malloc(len + 1);
    int j = 0;
    // 从后向前遍历，跳过空格，找到单词末尾再向前截取
    for (int i = len - 1; i >= 0; i--) {
        if (s[i] == ' ') continue;
        // i 当前是单词最后一个字符
        int wordEnd = i;
        while (i >= 0 && !isspace(s[i])) i--;
        // i 停在单词前空格，单词区间 [i+1, wordEnd]
        for (int p = i + 1; p <= wordEnd; p++) {
            t[j++] = s[p];
        }
        t[j++] = ' ';
    }
    if (j > 0) j--; // 删除末尾多余空格
    t[j] = '\0';
    return t;
}
```
