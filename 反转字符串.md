# Leetcode 344 反转字符串
https://leetcode.cn/problems/reverse-string

## 方法一 双指针
分别指向头尾，头尾字符交换，指针向中间移动。
### 易错点
while循环条件left < right。
### 代码
```c
void reverseString(char* s, int sSize) {
    int left = 0, right = sSize - 1;
    while (left < right)
    {
        char k = s[left];
        s[left] = s[right];
        s[right] = k;
        left++;
        right--;
    }
}
```
