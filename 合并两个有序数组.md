# Leetcode 88 合并两个有序数组
https://leetcode.cn/problems/merge-sorted-array

##方法一 暴力解法 插入后排序
把第二个数组中的元素先插入到第一个的末尾，再比较排序。
### 代码
```c
void merge(int* nums1, int nums1Size, int m, int* nums2, int nums2Size, int n) {
    for (int i = 0; i < n; i++)
    {
        nums1[m + i] = nums2[i];
        for (int j = m + i; j > 0; j--)
        {
            if (nums1[j - 1] > nums1[j])
            {
                int k = nums1[j - 1];
                nums1[j - 1] = nums1[j];
                nums1[j] = k;
            }
            else
            {
                break;
            }
        }
    }
}
```
