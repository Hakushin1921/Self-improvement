# Leetcode 88 合并两个有序数组
https://leetcode.cn/problems/merge-sorted-array

## 方法一 暴力解法 插入后排序
把第二个数组中的元素先插入到第一个的末尾，再比较排序。
### 代码一
```c
void merge(int* nums1, int nums1Size, int m, int* nums2, int nums2Size, int n) {
    for (int i = 0; i < n; i++)
    {
        nums1[m + i] = nums2[i];
        for (int j = m + i; j > 0; j--)
        {
            if (nums1[j - 1] > nums1[j])
            {
                int k = nums1[j - 1];
                nums1[j - 1] = nums1[j];
                nums1[j] = k;
            }
            else
            {
                break;
            }
        }
    }
}
```
### 代码二 快速排序
```c
int cmp(int* a, int* b) {
    return *a - *b;
}

void merge(int* nums1, int nums1Size, int m, int* nums2, int nums2Size, int n) {
    for (int i = 0; i != n; ++i) {
        nums1[m + i] = nums2[i];
    }
    qsort(nums1, nums1Size, sizeof(int), cmp);
}
```

## 方法二 双指针
用到了两个数组是有序的，只需要比较末尾的两个元素大小，把较大的放到第一个数组的末尾即可。这个方法不会丢失信息。
### 易错点
while循环的边界条件是两个数组的所有元素都遍历过一遍，但两指针之一超出数组的情况需要特别处理。
### 代码
```c
void merge(int* nums1, int nums1Size, int m, int* nums2, int nums2Size, int n)
{
    int ans1 = m - 1, ans2 = n - 1, ans = m + n - 1;
    while (ans1 >= 0 || ans2 >= 0)
    {
        if (ans1 == -1)
        {
            nums1[ans--] = nums2[ans2--];
        }
        else if (ans2 == -1)
        {
            nums1[ans--] = nums1[ans1--];
        }
        else if (nums1[ans1] <= nums2[ans2])
        {
            nums1[ans--] = nums2[ans2--];
        }
        else
        {
            nums1[ans--] = nums1[ans1--];
        }
    }
}
```
