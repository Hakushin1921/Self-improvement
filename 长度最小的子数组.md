# Leetcode 209 长度最小的子数组
https://leetcode.cn/problems/minimum-size-subarray-sum

## 滑动窗口
双指针，右指针加，左指针减，满足条件就进行一次判断。
### 易错点
1.右指针用for循环比while循环好，遍历到最后一个元素时左指针依旧可以移动。
2.min初始化设置的比数组长度大一个，防止出现子数组刚好是原数组的情况。
### 代码
```c
#define MIN(a, b) (a < b) ? a : b
int minSubArrayLen(int target, int* nums, int numsSize) {
    int len = 0, min = numsSize + 1, left = 0, sum = 0;
    for (int right = 0; right < numsSize; right++)
    {
        sum += nums[right];
        len++;
        while (sum >= target)
        {
            min = MIN(min, len);
            sum -= nums[left++];
            len--;
        }
    }
    if (min == numsSize + 1)
    {
        return 0;
    }
    else
    {
        return min;
    }
}
```
