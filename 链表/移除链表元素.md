# Leetcode 203 移除链表元素
https://leetcode.cn/problems/remove-linked-list-elements

## 遍历
遍历每个链表结点，判断结点的下一个结点值是否是目标值，如果是，则将当前结点指向下下个结点，并释放下一个结点空间。灵府山大佬的代码。
### 易错点
1.本题是不含头结点的单链表，所以要么单独判断头结点，要么手动添加一个空头结点（哨兵结点）指向第一个结点。  
2.指针的运用很频繁，注意对指针的理解。  
### 代码
```c
#include <cstdlib>
struct ListNode* removeElements(struct ListNode* head, int val) {
    struct ListNode dummy = {0, head};
    struct ListNode* cur = &dummy;
    while (cur->next)
    {
        struct ListNode* nxt = cur->next;
        if (nxt->val == val)
        {
            cur->next = nxt->next;
            free(nxt);
        }
        else
        {
            cur = cur->next;
        }
    }
    return dummy.next;
}

```
