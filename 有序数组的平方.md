# Leetcode 977 有序数组的平方
https://leetcode.cn/problems/squares-of-a-sorted-array

## 解法一 双指针
头尾两个指针，把平方大的放入输出数组的末尾，然后向内移动指针，最小元素一定在原数组中间位置。
### 易错点
此题要求用malloc申请返回数组的内存空间，不能直接覆盖输入，复用输入容器，要求自己开辟输出内存。
给 * returnSize 赋值
C 语言函数只能返回一个指针，没法同时返回两件事：
1.结果数组的地址  
2.结果数组有多少个元素  
LeetCode 的统一解决方案：开一个 int* returnSize 指针参数，让你在函数内部手动赋值 *returnSize = 长度，后台判题程序通过这个指针拿到数组长度，再遍历校验数组内容。
### 代码
```c
#include <cstdlib>

int* sortedSquares(int* nums, int numsSize, int* returnSize) {
    int* retnums =(int *) malloc(sizeof(int) * numsSize);
    *returnSize = numsSize;
    int left = 0, right = numsSize - 1, ans = numsSize - 1;
    while (left <= right)
    {
        int lx = nums[left] * nums[left];
        int rx = nums[right] * nums[right];
        if (lx > rx)
        {
            retnums[ans--] = lx;
            left++;
        }
        else
        {
            retnums[ans--] = rx;
            right--;
        }
    }
    return retnums;
}

```
