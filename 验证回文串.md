# Leetcode 125 验证回文串
https://leetcode.cn/problems/valid-palindrome

## 双指针
头尾两个指针，在不是字母数字时移到下一位，直到是数字为止，判断两个是否相等。
### 易错点
1.判断相等后要同时移动两个指针。
2.循环的边际条件是left < right，若两指针相差一位，仍需要继续判断，直到两指针重合，或者右指针小于左指针（aba,abba）
3.空字符串需要单独判断
### 代码
```c
#include <cstring>
#include <locale>

bool isPalindrome(char* s) {
    int left = 0, right = strlen(s) - 1;
    while (left < right)
    {
        if (! isalnum(s[left]))
        {
            left++;
            continue;
        }
        if (! isalnum(s[right]))
        {
            right--;
            continue;
        }
        if (tolower(s[left]) == tolower(s[right]))
        {
            left++;
            right--;
        }
        else
        {
            return false;
        }
    }
    return true;
}

```
