# Leetcode 27 移除元素
[leetcode.cn/problems/remove-element](https://leetcode.cn/problems/remove-element)

## 方法一（非原地解法）
定义一个和给出数组一样大的数组，用for循环遍历原数组，值等于val的跳过，不等于的赋值给新数组，最后用新数组覆盖原数组。
### 易错点：
释放空间，养成好习惯。注意此解法非原地，是后面几种解法的退化。
### 代码
```c
#include <cstdlib>
int removeElement(int* nums, int numsSize, int val) {
    int* ret = (int*) malloc(sizeof(int) * numsSize);
    int index = 0, k = 0;
    for (int i = 0; i < numsSize; i++)
    {
        if (nums[i] != val)
        {
            ret[index++] = nums[i];
        }
        else
        {
            k++;
        }
    }
    for (int i = 0; i < index; i++)
    {
        nums[i] = ret[i];
    }
    free(ret);
    return k;
}
```

## 方法二 双指针（原地）
用双指针，一个指向读取信息的位置，一个指向修改信息的位置，保证读取信息速度>=修改信息的速度即可。
### 代码
```c
int removeElement(int* nums, int numsSize, int val) {
    int index = 0, k = 0;
    for (int i = 0;i < numsSize;i++)
    {
        if (nums[i] != val)
        {
            nums[index++] = nums[i];
            k++;
        }
    }
    return k;
}
```
